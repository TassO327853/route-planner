# 行程规划助手

门店工作计划 + 路线优化 PWA 应用。

## 功能

- 📅 **月历视图** — 标记工作日/非工作日/其他安排
- 🏪 **门店管理** — 录入门店名称、地址、任务、截止日期
- 🤖 **自动排班** — 按截止时间优先级自动分配门店到工作日
- 🗺️ **路线优化** — 最近邻 + 2-opt 算法，计算最短访问路线
- 📱 **PWA 安装** — 手机浏览器打开后"添加到桌面"，像 App 一样使用

## 技术栈

- Vue 3 + Vant 4（移动端 UI）
- Dexie.js（IndexedDB 本地存储）
- 高德地图 JS API（地址→坐标地理编码）
- 纯前端，无需服务器，数据存在浏览器本地

## 使用方式

1. 手机浏览器打开应用 URL
2. 点击浏览器菜单 → "添加到桌面"
3. 桌面出现"行程规划"图标，点开即用

## 本地开发

直接用浏览器打开 `index.html` 即可（需要联网加载 CDN 资源和高德地图 API）。

## 部署

本项目为纯静态文件，可部署到 GitHub Pages / Netlify / Vercel 等任意静态托管平台。
