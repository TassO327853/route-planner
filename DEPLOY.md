# 手动部署到 GitHub Pages

## 步骤 1：创建 GitHub 仓库
1. 打开 https://github.com/new
2. Repository name: `route-planner`
3. 勾选 **Public**
4. 不要勾选 "Add a README"（我们已有）
5. 点 **Create repository**

## 步骤 2：上传文件
1. 在新仓库页面，点 **"uploading an existing file"** 链接
2. 把 route-planner.zip 解压后的**所有文件**拖进去上传
3. 提交信息填：`feat: 行程规划 PWA v1.0`
4. 点 **Commit changes**

## 步骤 3：开启 GitHub Pages
1. 仓库页面 → **Settings**（右上角齿轮图标）
2. 左侧菜单 → **Pages**
3. Source 选 **Deploy from a branch**
4. Branch 选 **master**，文件夹选 **/ (root)**
5. 点 **Save**

## 步骤 4：访问应用
等 1-2 分钟后，访问：
```
https://TassO327853.github.io/route-planner/
```

手机浏览器打开这个链接 → 点浏览器菜单 → "添加到桌面" → 完成！
